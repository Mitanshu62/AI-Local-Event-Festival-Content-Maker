QUICK ANSWER: How to Deploy This Project?
------------------------------------------

OPTION 1: AUTOMATED DEPLOYMENT (Easiest)
-----------------------------------------
1. Complete prerequisites (see below)
2. Run: deploy.bat
3. Follow the prompts
4. Done!

OPTION 2: MANUAL DEPLOYMENT
----------------------------
1. Deploy Backend:
   cd backend
   serverless deploy

2. Deploy Frontend:
   cd frontend
   vercel --prod

OPTION 3: TRADITIONAL SERVER
-----------------------------
See DEPLOYMENT.md for VPS/Docker deployment


PREREQUISITES (Must Complete First!)
================================================================================

1. AWS ACCOUNT
   - Sign up: https://aws.amazon.com
   - Enable Bedrock (Claude AI)
   - Get access keys

2. VERCEL ACCOUNT
   - Sign up: https://vercel.com (free)

3. INSTALL TOOLS
   - Node.js 18+: https://nodejs.org
   - AWS CLI: https://aws.amazon.com/cli/
   - Serverless: npm install -g serverless
   - Vercel CLI: npm install -g vercel

4. CONFIGURE AWS
   aws configure
   (Enter your AWS access key and secret)


DETAILED STEP-BY-STEP
================================================================================

STEP 1: AWS SETUP (20 minutes)
-------------------------------
1. Create AWS account at https://aws.amazon.com
2. Go to IAM → Users → Create user
3. Attach these policies:
   - AWSLambdaFullAccess
   - AmazonAPIGatewayAdministrator
   - AmazonDynamoDBFullAccess
4. Create access key
5. Save Access Key ID and Secret

6. Configure AWS CLI:
   aws configure
   AWS Access Key ID: [your key]
   AWS Secret Access Key: [your secret]
   Default region: us-east-1
   Default output: json

7. Enable Bedrock:
   - Go to AWS Console → Bedrock
   - Click "Model access"
   - Enable "Claude 3 Sonnet"
   - Wait for approval


STEP 2: DEPLOY BACKEND (10 minutes)
------------------------------------
cd backend
npm install
serverless deploy

IMPORTANT: Copy the API Gateway URL from the output!
Example: https://abc123.execute-api.us-east-1.amazonaws.com/dev/api


STEP 3: DEPLOY FRONTEND (10 minutes)
-------------------------------------
cd frontend
npm install

# Create .env.local with your API URL
echo NEXT_PUBLIC_API_URL=https://your-api-url.amazonaws.com/dev/api > .env.local

# Login to Vercel
vercel login

# Deploy
vercel --prod

# Set environment variable in Vercel dashboard:
# 1. Go to vercel.com/dashboard
# 2. Select your project
# 3. Settings → Environment Variables
# 4. Add NEXT_PUBLIC_API_URL with your API URL
# 5. Redeploy: vercel --prod


STEP 4: TEST (5 minutes)
-------------------------
1. Open your Vercel URL
2. Select "Diwali" festival
3. Choose "English" language
4. Select "Friendly" tone
5. Click "Generate Content"
6. Verify content appears

DONE! Your app is live!


TROUBLESHOOTING
================================================================================

Problem: "Bedrock access denied"
Solution: Enable Bedrock in AWS Console → Bedrock → Model access

Problem: "Deployment failed"
Solution: Check AWS credentials with: aws sts get-caller-identity

Problem: "API request failed"
Solution: Verify NEXT_PUBLIC_API_URL in Vercel dashboard matches your API URL

Problem: "Module not found"
Solution: Delete node_modules and run npm install again
